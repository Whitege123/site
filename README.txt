3D 河内塔游戏 - 使用说明
=====================================
一、玩法
1. 通过滑块选择 3-8 个圆盘
2. 点击柱子顶部圆盘 → 选中
3. 点击目标柱子 → 放置
4. 违反规则会出现红色警告
5. 全部移到最右侧柱子 → 胜利
6. 自动解题：一键自动完成
7. 重置：恢复初始状态

二、使用的数组方法
- push() / pop()：模拟栈移动圆盘
- forEach()：渲染柱子与绑定事件
- from()：生成初始圆盘数组
- at(-1)：获取栈顶元素
- length：校验栈高度

三、技术特点
- 纯 HTML/CSS/JS，无外部库
- 3D 视觉效果：perspective、rotateX、translateZ
- 响应式布局，支持手机/平板/电脑
- 动画：警告抖动、胜利发光、圆盘悬浮
- 完整规则校验
